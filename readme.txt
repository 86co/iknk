cube.py

ルービックキューブは、任意の回転操作を何回も繰り返すといつかは必ず元の状態に戻るといいます。
これを実証しました。

ex) R U' F2
回転記号をスペース区切りで入力すると、元の状態に戻るまでに繰り返す回数が出力されます。
